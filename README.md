# Monty

Monty is an interpreter for Monty ByteCode files. Monty ByteCode files contain instructions for a simple stack-based virtual machine.

## Description

Monty ByteCode is a scripting language that is first compiled into Monty byte codes. Monty interprets these byte codes to perform stack operations. The interpreter executes byte codes line by line.

## Usage

To run the Monty interpreter, use the following command:

```sh
./monty file
